# C++
# Cout and Cin

```cpp
#include <iostream>
```

## Cout

Used to print.

```cpp
std::cout << "Hello world!" << std::endl; // endl is end line
// Hello world!
```

## Cin

Used to get input from user and enter inside of variable.

```cpp
int x = 0;
std::cin >> x;
std::cout << "You entered " << x << std::endl;

// You entered <Your input>
```
